Elementary Math Worksheet Generator

My elementary-aged daughters are working hard on their math facts (addition, subtraction, multiplication, division), but their school doesn't ever want to send homework. To supplement their learning, I created this app using React so that I can quickly generate practice sheets. I'm particularly proud of the form validation, which automatically detects all impossible scenarios based on the constraints. Code can be viewed on Github

Catch Crabs

Catch Crabs is an Android mobile game developed with Apache Cordova. It was inspired by an assignment that I gave to my AP Computer Science students, in which the user clicks an object that keeps moving around the screen. I decided to create Catch Crabs as a learning project. I learned so much!

  • Cross-platform development with Apache Cordova
  • MVC architecture
  • App store publishing
  • SVG
  • Web audio
  • Git source control
  • Browser targeting
  • github.com/matthew-stockinger/catch-crabs

2020 Advent of Code

I am proud of my solutions to the 2020 Advent of Code. Each year, the Advent of Code attracts tens of thousands of coders to partake in up to 50 computational challenges. I solved every challenge on my own. Only 7% of participants achieved this. I learned:
