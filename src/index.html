Catch Crabs is my first mobile app. It was inspired by an assignment that I gave to my AP Computer Science students, in which the user clicks an object that keeps moving around the screen. I decided to create Catch Crabs as a learning project. I learned so much!

  • MVC architecture

  • Web audio
  • Git source control
